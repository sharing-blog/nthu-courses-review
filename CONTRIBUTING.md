## Rules

### How to Contribute
1. Fork the repository.
2. Make your changes in a new branch.
  - Branch name: `add-CourseID`
  - e.g. `add-GE13579`
3. Follow [**Conventional Commit**](https://www.conventionalcommits.org/en/v1.0.0/).
4. Submit a Pull Request for review.
